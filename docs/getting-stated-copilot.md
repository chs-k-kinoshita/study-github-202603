# GitHub Copilotの始め方
- GitHub Copilot＝開発特化のAIコーディングアシスタント
  - 他の"Copilot"(Microsoft 365 Copilot等)と混同しないこと
- VSCodeの拡張「GitHub Copilot Chat」をインストールして自分のGitHubアカウントにブラウザ経由でサインインするだけで使える
- 下記の公式ExerciseをやればGitHub Copilotの使い方については一通りイメージ掴めるはず（個人的にはこれが一番の近道かと思っている）
  - [公式Exercise(getting-started-with-github-copilot)](https://github.com/skills/getting-started-with-github-copilot)
    - IssueとGitHubActionsで対話的に進む（ただし英語）
    - Freeプランで実施可能
  - GitHub Foundations用の公式トレーニング([GitHub Copilot の概要](https://learn.microsoft.com/ja-jp/training/modules/introduction-to-github-copilot/)の一部です)
- 公式ドキュメントへのリンク
  - [GitHub Copilot Get-Started(GitHub公式)](https://docs.github.com/ja/copilot/get-started/quickstart)
  - [VS Code での GitHub Copilot(VSCode公式)](https://vscode.dokyumento.jp/docs/copilot/overview)
- 有料プランにすることで
  - 制限の上限がFree版より拡大（定額分の制限超えは必要に応じて従量課金）
  - 強力なモデルを利用できるようになったり、高度な機能（AIをコードレビュアーにする等）が利用可能になる
